# 🚧 FIX BLR — Report Potholes, Fix Bengaluru

A free civic platform where Bengaluru citizens can report potholes 
with photos and GPS coordinates — automatically emailing complaints 
to BBMP and Karnataka Urban Development Department.

## 🌐 Live Website
👉 https://naren0316.github.io/fixblr-civic-engine/

## 💡 Features
- 📸 Upload 3 photos from different angles
- 📍 Auto GPS location detection
- 📧 Auto-sends complaint email to BBMP
- 🗺️ Live Bengaluru pothole map
- 📄 PDF proof of complaint generated

## 🛠️ Built With
HTML · CSS · JavaScript · EmailJS · Leaflet.js
